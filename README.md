# Анализатор звука двигателя автомобиля

Приложение для анализа аудиофайлов записи звука двигателя. Отображает АЧХ, спектрограмму и другие характеристики для диагностики.

## Возможности

- Загрузка MP3, WAV, FLAC, OGG, M4A файлов
- Отображение временного сигнала
- **АЧХ (Амплитудно-частотная характеристика)**
- **Спектрограмма**
- **MFCC коэффициенты** (анализ тембра)
- Анализ доминирующих частот
- Оценка распределения энергии по частотным диапазонам
- Рекомендации по диагностике двигателя

## Установка

### 1. Активация виртуального окружения

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск приложения

```bash
python sound_analyzer.py
```

## Использование

1. Нажмите кнопку **"Загрузить MP3/WAV"**
2. Выберите аудиофайл с записью звука двигателя
3. Нажмите **"Анализировать"**
4. Изучите графики и информацию о характеристиках звука

## Интерпретация результатов

### Частотные диапазоны для двигателя:

| Диапазон | Частоты | Значение |
|----------|---------|----------|
| НЧ | 0-100 Гц | Холостые обороты, базовая частота вращения |
| СЧ | 100-1000 Гц | Рабочие обороты, основная работа двигателя |
| ВЧ | 1000-5000 Гц | Шумы, возможные дефекты (подшипники, клапаны) |

### Признаки проблем:

- **Высокий уровень ВЧ (>30%)** — возможные механические дефекты
- **Пики на специфических частотах** — дисбаланс, износ подшипников
- **Широкополосный шум** — проблемы с выхлопной системой
